# Claw 错题管理系统 - 使用说明

## 快速入门

### 访问系统

1. 确保后端服务已启动（参见部署指南）
2. 打开浏览器访问 `http://localhost:8500/app`
3. 或启动桌面客户端 `python desktop/main.py`

## Web 界面使用

### 一、上传处理

1. 点击上传区域或拖拽图片文件到上传区域
2. 支持的格式：JPG、PNG、BMP、WebP、TIFF
3. 文件大小限制：默认 50MB（可在配置中修改）
4. 系统将自动执行以下流程：
   - 上传文件到服务器
   - 提交 PP-StructureV3 异步任务（一步完成版面分析、OCR识别、表格/公式识别）
   - 轮询等待任务完成
   - 下载并解析 Markdown / JSON 结果
   - 生成结构化报告

### 二、查看处理进度

处理过程中会显示实时进度：
- **上传文件**：将图片传输到服务器
- **提交任务**：提交至 PP-StructureV3 API 进行文档结构化分析
- **处理中**：PP-StructureV3 一步完成版面分析 + OCR 识别 + 表格/公式识别
- **下载结果**：获取 Markdown / JSON 结构化结果
- **生成报告**：生成 Markdown 格式报告

### 三、查看结果

处理完成后显示：
- **原始图片** vs **版面分析图片** 对比
- **处理统计**：耗时、识别页数、状态
- **Markdown报告**：结构化的识别结果

### 四、处理记录

在"处理记录"标签页查看所有历史处理：
- 处理编号、文件名、时间
- 处理状态（成功/失败）
- 处理耗时和识别页数
- 点击"查看"可查看详细报告

### 五、报告中心

在"报告中心"标签页管理所有生成的报告：
- **查看详情**：查看完整 Markdown 报告
- **下载**：下载报告文件（.md格式）
- **删除**：删除不需要的报告

### 六、系统配置

在"系统配置"标签页进行参数设置：

#### PaddleOCR API 配置
- **API 地址**：PaddleOCR 服务地址
- **API Token**：认证 Token（从百度AI Studio获取）
- **测试连接**：验证 API 配置是否正确

#### 服务器配置
- **监听地址/端口**：后端服务绑定参数
- **最大上传大小**：限制上传文件大小

#### 处理参数
- **日志级别**：控制日志详细程度

## 桌面客户端使用

桌面客户端提供与 Web 界面相同的功能：

### 主界面
- **上传处理**：选择文件并处理
- **处理记录**：查看历史记录
- **报告中心**：管理报告
- **系统配置**：配置参数

### 快捷操作
- 菜单栏：文件 → 打开图片
- 工具栏：刷新所有数据
- 状态栏：显示当前服务器连接状态

## Markdown 报告格式

生成的报告包含以下部分：

```markdown
# 错题分析报告
- 生成时间、原始文件、文档编号

## 处理信息
- 处理耗时、识别页数

## OCR识别结果
- 题号、题目内容、选项、参考答案、解析

## 版面分析图片
- PP-StructureV3 版面分析结果图

## OCR原始数据
- JSON格式的完整识别结果
```

## 常见问题

### Q: 上传后处理失败？
A: 检查以下几点：
- PaddleOCR API 密钥是否已配置
- API 服务是否可访问
- 图片格式是否支持
- 查看 `logs/error.log` 获取详细错误信息

### Q: OCR识别结果不准确？
A: 尝试以下调整：
- 提高上传图片质量
- 尝试切换模型（PaddleOCR-VL-1.5 / PP-StructureV3）
- 确保图片文字清晰可辨

### Q: 如何备份数据？
A: 备份以下目录：
- `output/` - 所有生成的报告
- `uploads/` - 上传的原始图片
- `logs/` - 系统日志

### Q: 如何修改端口？
A: 修改 `.env` 中的 `PORT` 配置，或在 Web 界面的系统配置中修改。
